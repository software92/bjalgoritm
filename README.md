# 백준 알고리즘 문제풀이

- 자바스크립트를 사용해 문제풀이
- '단계별로 문제풀이'에서 6단계부터 시작
- 1~5단계에서 누락한 항목만 문제풀이
- 파일이름은 (날짜-문제번호).js로 저장
- 매일 한개씩이라도 작성
